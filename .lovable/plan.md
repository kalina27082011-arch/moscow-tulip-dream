4. «Лимонад» — 45 жёлтых с веточками мимозы
5. «Акварель» — 49 пастельный микс (пудра, персик, лаванда)

Для каждого: имя, slug, цена, описание (1-2 романтичных предложения), состав (количество и сорт), color_tag, sort_order, фото.

### Технические детали

1. **Генерация фото** — `imagegen--generate_image` (fast), сохраняю в `src/assets/tulips-<slug>.jpg`, единый стиль: студийная съёмка, мягкий дневной свет, нейтральный фон (крем/льняная ткань), вертикальный 3:4, чтобы вписаться в существующую сетку каталога.

2. **Реестр изображений** — дополняю `src/lib/product-images.ts`: импорты новых файлов и записи в `map`. Поле `image_url` в БД хранит имя файла (`tulips-<slug>.jpg`), резолвер уже подтягивает локальные ассеты.

3. **Миграция данных** — `supabase--insert` с одним `INSERT INTO public.products (...) VALUES (...), (...), ...` для 12 строк. `is_active=true`, `sort_order` начиная с 100, чтобы не перебивать существующие.

4. **Проверка** — после билда открыть `/catalog` через Playwright, убедиться что новые карточки видны, фильтры по цветам работают, картинки грузятся.

Существующие 7 букетов и админка не трогаются — только добавление.
